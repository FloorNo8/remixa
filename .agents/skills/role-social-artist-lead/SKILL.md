# Job Description: Social Artist Co-Creation Lead (role-social-artist-lead)

## Objective
Establish database structures and API features to enable viral co-creation, automatic royalty split tree distribution, and social publishing tracking.

## Scope of Work
1. **Co-Creation Lineage Tree:** Expose lineage API endpoints mapping parent/grandparent nodes, active derivative work branches, and cumulative views.
2. **Royalty Splits Auditing:** Work with the Royalty Steward to ensure exact 40/30/30 splits (30% parent / 10% grandparent) are enforced.
3. **Sharing Virality Hooks:** Track social shares and referrals to measure and optimize short-form sound propagation loops.
