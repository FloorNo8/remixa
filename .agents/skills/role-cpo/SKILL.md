# Job Description: Chief Product Officer (role-cpo)

## Objective
Design the pricing models, subscription tiers, usage quotas, and user experience flows to achieve a minimum **35% gross profit margin** across all platform activities.

## Scope of Work
1.  **Monetization Tiers:** Define standard and premium subscription plans (Free, Pro, Business) and developer API tiers.
2.  **Product Quotas:** Establish hourly and monthly limits for track generations, remixes, and whitelisted videos based on compute cost profiles.
3.  **Telemetry Specifications:** Define the data fields for tracking usage metrics (generations/hour, remixes/hour, whitelists/month) and user cost metrics.
4.  **Reach Score Design:** Structure the logarithmic "Reach Score" algorithm to highlight commercial utility for creators.
