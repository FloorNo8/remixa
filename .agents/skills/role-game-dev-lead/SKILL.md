# Job Description: Game Developer API Lead (role-game-dev-lead)

## Objective
Design and implement developer-facing APIs that expose separated stem tracks and allow runtime parameter variations to support interactive audio integrations in video games.

## Scope of Work
1. **Stems API:** Implement the endpoint to retrieve individual stem layer tracks (drums, vocals, synths, base).
2. **Interactive Audio Metadata:** Expose active production parameters (drive, high-shelf, sub-bass, stereo width) associated with styles and producer personas.
3. **Dynamic Score Control:** Support queries that adapt playback options based on client-defined gameplay levels.
